## 🛠 How to Use

1. **For AI Agents:** When working with `Gemini-CLI`, `Cursor`, or `Copilot`, provide the path to the relevant file (e.g., `frontend/JavaScript/readme.md`) in the system prompt or chat context.
2. **For Code Review:** Use these files as a reference point when reviewing Pull Requests to ensure industry-standard implementation.
3. **For Learning:** Study the deep technical arguments (JIT optimizations, memory leaks, V8 internals) to understand "why" a practice is chosen, not just "how" to write it.
